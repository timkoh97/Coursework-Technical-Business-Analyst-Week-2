# Current State Analysis

## A customer

This process is drawn-out, asks for data already given and unclear, not to mention inconsistent. SMS reminders may be sent for payments already promised, for example.

## An agent

This process requires lots of manual checking which is slow, can reveal conflicting data which takes time to reconcile and often results in missing a follow-up. It is exhausting and high risk.

## A team leader

This process is opaque, with little clarity on the work being done, and reactive, with valuable time being used to distinguish high and low complexity cases from unreconciled data.
